# Crawl from NewEgg.com
## Using BeautifulSoup
1. install package
'''
pip install pandas bs4 requests requests_html
'''
2. Run the scripts 
## Using Selenium
1. install package selenium
'''
pip install selenium
'''
2. install your browser driver (in this code, we use Chrome)
You can install your chrome driver version here: 
https://chromedriver.chromium.org/downloads
3. Run the scripts
